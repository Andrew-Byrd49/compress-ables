# Compress-able

A 100% client-side image compression tool built with React and Tailwind CSS.

## Features
- **Privacy First**: Images never leave your browser.
- **Modern Formats**: Convert to WebP and AVIF.
- **Batch Processing**: Compress multiple images at once.
- **Client-Side ZIP**: Download all processed images as a single ZIP file.
- **Customizable**: Adjust quality, resize, and format.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Compressor.js
- JSZip
- Lucide React

## Development
1. `npm install`
2. `npm run dev`

## Deployment
Build for production:
`npm run build`

The output `dist` folder can be hosted on any static hosting service (Netlify, Vercel, GitHub Pages).
